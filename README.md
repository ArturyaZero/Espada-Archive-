# 🗡️ Espada Archive

A modern and responsive character showcase inspired by **Bleach**.
This project presents the **Espada** in an interactive card slider built with **HTML, CSS, JavaScript, and Swiper.js**.

## ✨ Features

* 🎴 Responsive card slider
* ⚡ Smooth animations powered by Swiper.js
* 🎨 Dark UI inspired by Hueco Mundo
* 🌈 Unique color theme for each Espada
* 📱 Mobile-friendly design
* 🖼️ Character artwork with custom badges

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Swiper.js
* Google Fonts

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/espada-archive.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.
